## Run Locally

1. Clone the project

```bash
  git clone https://github.com/Chipl7/StringPlus.git
```

2. Go to the project directory

```bash
  cd StringPlus
```

3. Src dir

```bash
  cd src
```

4. Run make for build

```bash
  make all
```
